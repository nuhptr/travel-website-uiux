# Next.js + Tailwind CSS + TypeScript Travel Camp Website

This is a travel camp website built with Next.js, Tailwind CSS, and TypeScript.

<!-- Image Section -->

## Getting Started

-  Clone the repo
-  Run `npm install`
-  Run `npm run dev`
